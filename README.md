### Что делает:
- Полная автоматизация генерации обложек (сохраняются на всякий).
- Публикация на Rutube с сохранением ключей трансляции в отдельный файл.
- Автоматическое прикрепление ссылок на сайт Footballista.

### Инструкция по запуску:
1. Скачайте файл GOOAAL.exe ниже и положите в отдельную папку.
2. Запустите его.
3. Нажмите кнопку "1. Открыть Chrome..." и залогиньтесь во все сервисы.
4. Откройте [Footballista](https://footballista.ru/admin/games) на вкладке 'Игры', [AFL Graphics](https://afl-graphics.vercel.app/igraphics/video), [Rutube](https://studio.rutube.ru/) с правильным аккаунтом.
5. Запустите.
6. Следите за логами

ЗЫ. Нужно иметь скачанный Chrome
